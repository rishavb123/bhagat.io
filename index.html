About Me

Hello. My name is Rishav Bhagat. I am currently attending the South Brunswick High School where I help run the Computer Science Club. We host HackSB at the South Brunswick Public Library. I am extremely interested in computer science, physics, and mathematics. To see some of my best projects scroll down or check on my github.

Projects

Neural Network

This is an artificial neural network that I have created from scratch using my math library in java. I have used it to recognize hand written digits using the mnist dataset and even classify certain flowers. It uses concepts of supervised learning and stochastic gradient dissent.

Java Math Library

This is java library that can do a lot of math including concepts from statistics, linear algebra, and calculus. It is mostly focused on linear algebra for now since that has to most application, but I will add more to it.

Home Server

This is a server running in my home made using nodejs and express that I made to host files, control some of the smart devices in my house, and even send me emails when needed.

Smart Room

This serves a react app that connects to my Home Server to control some of the smart home devices in my house (mainly in my room). It makes them voice controlled rather than only controlled by my phone.
